# IdeaStocker
IdeaStockerは、将来やりたいことや、漠然としたことなど自由にテーマを決めて、思いついたタスクやひらめきなどメモして管理できるアプリケーションです。

# URL
https://appy-nuxt-rails-front.herokuapp.com/

# 使用技術

- フロントエンド
  - Nuxt.js 2.15.7
  - Vuetify 2.5.10

- バックエンド
  - Ruby on Rails (APIモード) 6.0.4

- インフラ
  - Heroku
  - Docker 20.10.5
  - Dokcer-compose 1.29
- 開発PC macOS 10.13
